Binance Butler

*Unofficial* Assistant to help you get ahead with your trades on binance.com
  1. Order fill notifications :
    With volatility of crypto market, order fill notifications are vital for traders. Butler notifies when an order is filled. Butler runs in background so you don't have to constantly check order screen.
  2. Intuitive portfolio tracker : Take a quick peek how your portfolio is doing with one click on app icon. Tracker also provide percentage breakdown and $ worth of each coin in same screen.

How to generate API key

You need API key as QR code from binance to access this app.
